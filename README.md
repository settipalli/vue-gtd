# vue-gtd

Track tasks to be completed Today, Tomorrow, Later.


## Project setup
```
yarn install
```


### Compiles and hot-reloads for development
```
yarn serve
```


### Compiles and minifies for production
```
yarn build
```


### Lints and fixes files
```
yarn lint
```
